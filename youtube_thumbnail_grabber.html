    YouTube Thumbnail Grabber

    This is a simple utility to view and download all the JPEG and WEBP thumbnails for a YouTube video. As of August 2024, the web address structure of these images is fairly straight forward based upon the video's ID, but since it's tedious to manually typing in these addresses one-by-one, I build this simple utility to automatically load all the potential images at once.
