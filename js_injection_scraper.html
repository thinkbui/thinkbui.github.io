    JavaScript Injection Scraper

    This is a template for use to scrape e-Commerce webpages for its images (and possibly videos) to be assist online sellers. If, say, you wanted to sell something you no longer need on eBay and can only find a professional photo on someone else's listing, injecting a script like mine will allow you to download it so you can upload it on your own listing.

    As a warning, JavaScript injection is a common way for hackers to steal sensitive information or other malicious activity, so you should never inject code you haven't read through or understand into the page you're viewing and therefore I'm not including instructions on the actual injection to protect the novice. Veteran web developers should know how to do this.
